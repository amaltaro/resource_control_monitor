Workload Management Resource Control Monitor

This service provides real-time monitoring and metrics for workload management resources. It offers RESTful APIs to track system status, resource utilization, and performance metrics.

Available Endpoints:

  • /wm_resource_monitor/api/status - System status, version, and health information
  • /wm_resource_monitor/api/metrics - Prometheus-compatible metrics endpoint

Features:

  • Real-time resource monitoring
  • Prometheus metrics integration
  • RESTful API architecture
  • Secure X.509 certificate-based authentication
  • Compressed data transfer

For API documentation and integration details, please refer to /wm_resource_monitor/api
